# 🍳 AI 칼로리 계산기 — Replit 구현 지시서

## 프로젝트 개요

직업계고(조리·식품 전공) 학생들을 위한 **AI 음식 칼로리 분석 앱 프로토타입**.
바이브코딩 교육 프로그램의 데모 앱으로, 학생들이 "이런 앱을 직접 만들 수 있다"는 것을 보여주는 용도.

### 핵심 사용자 흐름

```
음식 사진 촬영/선택 → AI 분석 (1.5초 로딩) → 칼로리 결과 표시 → 기록 저장 → 히스토리 확인
```

### 기술 요구사항

| 항목 | 사양 |
|------|------|
| 프레임워크 | React (Vite) 또는 Next.js |
| 스타일링 | Tailwind CSS 또는 CSS Modules |
| AI API | OpenAI Vision API (gpt-4o-mini) |
| 이미지 | Unsplash 무료 이미지 (샘플용) + 실제 카메라/갤러리 업로드 |
| 배포 | Replit 내 배포 |
| 반응형 | 모바일 퍼스트 (390px 기준), 데스크탑은 폰 프레임으로 감싸기 |

### 중요 원칙

1. **모바일 퍼스트** — 학교 교실에서 스마트폰으로 사용
2. **실제 이미지 사용** — 이모지/아이콘이 아닌 진짜 음식 사진으로 신뢰감
3. **데모 + 실제 기능 병행** — 샘플 음식 6개는 하드코딩, 카메라 촬영은 실제 AI 분석
4. **한국어 UI** — 모든 텍스트 한국어

## 파일 구조 (권장)

```
/src
  /components
    Header.jsx          # 상단 헤더 (앱 타이틀 + DEMO 배지)
    BottomTabs.jsx      # 하단 탭 네비게이션 (촬영/기록)
    CaptureTab.jsx      # 📷 촬영 탭 메인
    HistoryTab.jsx      # 📊 기록 탭 메인
    FoodGrid.jsx        # 샘플 음식 6개 그리드
    LoadingOverlay.jsx  # AI 분석 로딩 화면
    ResultView.jsx      # 분석 결과 화면
    CalorieGauge.jsx    # 일일 섭취량 게이지 바
    NutritionBar.jsx    # 탄단지 비율 바
    FoodItemList.jsx    # 인식된 음식 개별 목록
    MealGroup.jsx       # 히스토리 시간대별 그룹
  /data
    sampleMeals.js      # 샘플 음식 6개 데이터
  /api
    analyzeFood.js      # OpenAI Vision API 호출
  /hooks
    useHistory.js       # 기록 상태 관리
  App.jsx
  index.css             # 글로벌 스타일 + CSS 변수
```
