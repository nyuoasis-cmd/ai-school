# 🚀 Replit 빠른 시작 프롬프트

아래 프롬프트를 Replit AI에 **그대로 복붙**하세요.
상세 스펙은 다른 문서들을 참고하면 됩니다.

---

## Replit에 붙여넣을 프롬프트

```
모바일 퍼스트 AI 칼로리 계산기 앱을 만들어줘.

[프로젝트 개요]
- 직업계고 조리·식품 전공 학생을 위한 데모 앱
- React (Vite) + Tailwind CSS
- OpenAI Vision API로 음식 사진 분석

[화면 구조]
하단 탭 2개: 📷촬영 | 📊기록

[📷 촬영 탭]
1. 히어로: "오늘 뭐 먹었어? 📸" + 설명 텍스트
2. 카메라/갤러리 버튼 2개 (가로 배치)
3. 체험용 샘플 음식 3x2 그리드 (실제 음식 사진, Unsplash 이미지 사용)
   - 한식 정식 685kcal, 크림 파스타 780kcal, 햄버거 세트 1050kcal
   - 비빔밥 580kcal, 샐러드 볼 320kcal, 라면 550kcal
4. 샘플 탭 → 1.5초 로딩 연출 → 결과 화면
5. 카메라/갤러리 → 실제 OpenAI API 호출 → 결과 화면

[로딩 화면]
- 음식 사진 + 프로그레스 바 + 단계별 텍스트 변화
- 0→35→70→100% (0ms, 150ms, 600ms, 1100ms)

[결과 화면]
- 상단: 음식 사진 크게 (16:10)
- 총 칼로리 (큰 숫자, 오렌지색)
- 일일 권장량 게이지 (0~2000kcal)
- 탄단지 비율 세그먼트 바 (오렌지/블루/옐로)
- 인식된 음식 개별 리스트 (썸네일 + 이름 + 칼로리)
- 하단: "다시 찍기" / "기록 저장" 버튼

[📊 기록 탭]
- 오늘 총 섭취량/2000kcal 게이지
- 시간대별 그룹: 🌅아침, ☀️점심, 🍪간식, 🌙저녁
- 각 그룹에 음식 카드 리스트

[디자인]
- 색상: bg #F7F7F5, accent #E85D2A, green #27AE60, blue #3B82C4, yellow #E5A020
- 폰트: Noto Sans KR
- 카드: radius 14px, shadow 0 1px 8px rgba(0,0,0,0.05)
- 애니메이션: fadeUp 등장, pulse 로딩, 게이지 fill 트랜지션
- 데스크탑: 390x844 폰 프레임으로 감싸기 (radius 44px, 노치 포함)
- 이모지가 아닌 실제 음식 사진 사용 (Unsplash)

[AI API]
- 모델: gpt-4o-mini
- 시스템 프롬프트: 음식 사진 → JSON 응답 (name, totalCal, carb%, protein%, fat%, foods 배열)
- 서버 라우트 /api/analyze 통해 호출
- API 키는 Replit Secrets에 OPENAI_API_KEY로 저장

[참고]
- 첨부된 calorie-app-v2.html이 완성된 UI 디자인 레퍼런스
- 이 HTML과 동일한 룩앤필로 React 컴포넌트 구현
```

## 파일 전달 체크리스트

Replit에 같이 전달할 파일들:

| 파일 | 용도 | 필수 |
|------|------|------|
| `calorie-app-v2.html` | UI 디자인 레퍼런스 (브라우저에서 열어볼 수 있음) | ✅ |
| `01_프로젝트_개요.md` | 전체 구조와 기술 스택 | ✅ |
| `02_화면별_UI_스펙.md` | 화면별 상세 레이아웃과 디자인 토큰 | ✅ |
| `03_데이터_API_스펙.md` | 샘플 데이터 + OpenAI API 연동 코드 | ✅ |

## 진행 순서 추천

1. Replit에서 React (Vite) 프로젝트 생성
2. 위 프롬프트 붙여넣기
3. `calorie-app-v2.html`을 프로젝트에 첨부하여 디자인 레퍼런스로 제공
4. 샘플 음식 하드코딩 먼저 동작 확인
5. OpenAI API 키 설정 → 실제 카메라 분석 기능 추가
6. 데스크탑 폰 프레임 적용
7. 최종 테스트
