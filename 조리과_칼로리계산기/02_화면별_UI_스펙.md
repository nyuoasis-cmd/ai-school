# 📱 화면별 UI 상세 스펙

## 디자인 토큰 (CSS 변수)

```css
:root {
  /* 배경/표면 */
  --bg: #F7F7F5;
  --surface: #FFFFFF;

  /* 텍스트 */
  --text: #1A1A1A;
  --text2: #555555;
  --text3: #999999;

  /* 테두리 */
  --border: #EEECE7;

  /* 브랜드 색상 */
  --accent: #E85D2A;         /* 메인 오렌지 */
  --accent-light: #FFF3ED;   /* 오렌지 배경 */

  /* 의미 색상 */
  --green: #27AE60;
  --green-light: #E8F8F0;
  --blue: #3B82C4;
  --blue-light: #EBF3FB;
  --yellow: #E5A020;
  --yellow-light: #FEF6E6;

  /* 영양소 색상 */
  --carb: #E85D2A;     /* 탄수화물 = 오렌지 */
  --protein: #3B82C4;  /* 단백질 = 블루 */
  --fat: #E5A020;      /* 지방 = 옐로 */

  /* 간격/크기 */
  --radius: 14px;
  --radius-sm: 10px;
  --shadow: 0 1px 8px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 20px rgba(0,0,0,0.08);
  --tab-height: 60px;
  --header-height: 52px;
}
```

### 폰트

- **Noto Sans KR** (Google Fonts)
- 굵기: 400, 500, 600, 700, 800

---

## 공통 레이아웃

```
┌─────────────────────────┐
│  [Header]  52px 높이     │  sticky top
├─────────────────────────┤
│                         │
│  [Tab Content]          │  스크롤 영역
│  padding-bottom: 80px   │
│                         │
├─────────────────────────┤
│  [Bottom Tabs]  60px    │  fixed bottom
└─────────────────────────┘
```

### Header
- 왼쪽: "AI 칼로리" (17px, 800) + `DEMO` 배지 (9px, 배경 accent, 흰색, radius 4px)
- 오른쪽: "조리·식품" (12px, border 배경, radius 20px)

### Bottom Tabs
- 2개 탭: 📷촬영 | 📊기록
- 배경: rgba(255,255,255,0.95) + backdrop-filter blur(16px)
- active 상태: accent 색상

## 화면 1: 📷 촬영 탭 (메인)

### 1-1. 히어로 영역
```
┌─────────────────────────┐
│   오늘 뭐 먹었어? 📸     │  24px, 800 weight
│   음식 사진을 찍으면      │  14px, text2 색상
│   AI가 칼로리를 분석해요  │
└─────────────────────────┘
```

### 1-2. 촬영 버튼 2개 (가로 배치)
```
┌───────────┐ ┌───────────┐
│   📷      │ │   🖼️      │
│ 사진 촬영  │ │ 갤러리선택 │
│ 카메라로   │ │ 저장된사진 │
└───────────┘ └───────────┘
```
- flex: 1/1, gap 10px
- 테두리: 1.5px solid border색
- :active 시 → scale(0.97), border accent색, 배경 accent-light

### 1-3. 체험 음식 그리드
- 제목: "💡 체험해보기" + "탭해서 분석" (우측)
- 3열 그리드, gap 10px
- 각 카드: aspect-ratio 1:1, radius 10px
  - 실제 음식 사진 (object-fit: cover)
  - 하단 오버레이: gradient(transparent → rgba(0,0,0,0.7))
  - 왼쪽: 음식명 (12px, 흰색, 600)
  - 오른쪽: 칼로리 (10px, 흰색 80%)
- :active 시 scale(0.95)

**6개 샘플 음식:**

| # | 이름 | 칼로리 | 이미지 |
|---|------|--------|--------|
| 0 | 한식 정식 | 685 kcal | 한식 상차림 사진 |
| 1 | 크림 파스타 | 780 kcal | 크림 파스타 사진 |
| 2 | 햄버거 세트 | 1,050 kcal | 치즈버거 사진 |
| 3 | 비빔밥 | 580 kcal | 비빔밥 사진 |
| 4 | 샐러드 볼 | 320 kcal | 샐러드 사진 |
| 5 | 라면 | 550 kcal | 라면 사진 |

## 화면 2: 로딩 오버레이

```
┌─────────────────────────┐
│                         │
│      [음식 사진]         │  120x120, radius 20px
│      (펄스 애니메이션)    │  scale(1) ↔ scale(1.03)
│                         │
│      ━━━━━━━━━━━━       │  프로그레스 바 (180px)
│                         │
│  AI가 음식을 분석하고     │  15px, 600
│     있어요               │
│    잠깐만요...           │  13px, text3
└─────────────────────────┘
```

### 로딩 시퀀스 (총 1.5초)
| 시점 | 진행률 | 텍스트 변화 |
|------|--------|------------|
| 0ms | 0% | "잠깐만요..." |
| 150ms | 35% | "음식을 인식하는 중..." |
| 600ms | 70% | "{첫번째 음식명} 발견!" |
| 1100ms | 100% | "분석 완료! / {N}가지 음식을 찾았어요" |
| 1500ms | — | 오버레이 닫기 → 결과 화면 |

- 배경: rgba(247,247,245, 0.97)
- 프로그레스 바: gradient(accent → yellow)

## 화면 3: 분석 결과

결과 화면은 촬영 탭 내에서 메인 화면을 숨기고 표시.

### 3-1. 음식 사진
```
┌─────────────────────────┐
│                         │
│    [음식 사진 크게]       │  aspect-ratio: 16/10
│                         │
│              AI 분석 결과 │  우상단 배지
└─────────────────────────┘
```

### 3-2. 총 칼로리 카드
```
┌─────────────────────────┐
│      총 칼로리            │  12px, text3
│       685 kcal           │  44px, 800, accent색
│  ─────────────────────   │
│  일일 권장    685/2,000   │
│  ━━━━━━━░░░░░░░░░░░░░   │  게이지 바 (8px)
│  0   500  1000 1500 2000 │
└─────────────────────────┘
```
- 게이지 색상: 0~90% → green, 90%+ → accent(경고)
- 누적 계산: 이전 기록 + 현재 음식

### 3-3. 영양소 비율 카드
```
┌─────────────────────────┐
│  영양소 비율              │
│  ████████████░░░░░░░░    │  세그먼트 바 (12px)
│  탄수화물  단백질   지방   │
│    55%      25%     20%  │
└─────────────────────────┘
```
- 3색 세그먼트 바: carb | protein | fat
- 비율에 따라 width% 조절

### 3-4. 인식된 음식 목록
```
┌─────────────────────────┐
│  🔍 인식된 음식 4가지     │
│  ┌────┐                  │
│  │사진│ 쌀밥              │
│  │    │ 1공기 (200g)      │  310 kcal
│  └────┘                  │
│  ─────────────────────   │
│  ┌────┐                  │
│  │사진│ 김치찌개           │
│  │    │ 1인분             │  200 kcal
│  └────┘                  │
│  ...                     │
└─────────────────────────┘
```
- 썸네일: 44x44px, radius 10px
- 칼로리: accent색, 15px, 700

### 3-5. 하단 버튼
```
┌───────────┐ ┌───────────┐
│ 📷 다시찍기│ │ ✅ 기록저장│
│  (회색)    │ │ (accent)  │

```
- "다시 찍기": 메인 화면으로 복귀
- "기록 저장": 히스토리에 추가 → 기록 탭으로 이동

## 애니메이션

| 이름 | 효과 | 용도 |
|------|------|------|
| fadeUp | opacity 0→1, translateY 14→0, 0.35s | 카드 등장 |
| pulse | scale 1↔1.03, opacity 1↔0.85, 1s | 로딩 이미지 |
| gauge-fill | width 0→N%, 0.8s ease | 게이지 바 채우기 |
| card-press | scale(0.95~0.97) | 카드/버튼 터치 |

- 결과 화면 카드들: 순차 등장 (0ms, 80ms, 160ms, 240ms 딜레이)
