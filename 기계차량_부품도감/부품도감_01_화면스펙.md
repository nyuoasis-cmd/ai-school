## 인터랙션 정리

| 요소 | 동작 | 효과 |
|------|------|------|
| 탭 전환 | 클릭 | 컨텐츠 교체 + 활성 인디케이터 |
| 카테고리 칩 | 클릭 | 활성 전환 + 카드 필터링 |
| 부품 카드 | 수평 스와이프 | 스냅 스크롤 + 도트/카운터 업데이트 |
| 상세보기 버튼 | 클릭 | 부품 상세 페이지 이동 |
| 핫스팟 | 클릭 | 활성 전환 + 패널 업데이트 (fadeInUp) |
| 퀴즈 선택지 | 클릭 | 정답/오답 색상 + AI 해설 표시 |
| 다음 문제 | 클릭 | 다음 퀴즈 로드 + 진행바 업데이트 |

## 애니메이션

| 이름 | 대상 | 효과 |
|------|------|------|
| fadeInUp | 일반 요소 진입 | opacity 0→1, Y +20→0, 0.5s |
| slideInRight | 카드 진입 | opacity 0→1, X +40→0, 0.5s (딜레이 순차) |
| hotspotPulse | 핫스팟 | scale 1↔1.1, 2s 무한 |
| hotspotRing | 핫스팟 외곽 링 | scale 1→1.8 + opacity 1→0, 2s 무한 |
| hintPulse | 스와이프 힌트 | opacity 0.5↔1, 2s 무한 |
