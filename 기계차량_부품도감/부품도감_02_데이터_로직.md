## 4. 로직 흐름

### 탭1: 부품 도감
```
앱 로드 → 전체 부품 8개 카드 렌더링
       → 카테고리 칩 클릭
       → 해당 카테고리 부품만 필터링
       → 카드 재렌더링 + 도트/카운터 리셋
       → 카드 스와이프
       → scroll 이벤트 → 현재 카드 인덱스 계산
       → 도트 인디케이터 업데이트
       → 카운터 텍스트 업데이트
```

### 탭2: 차량 구조
```
탭 전환 → SVG 차량도 + 핫스팟 5개 렌더링
       → 기본 활성: 엔진
       → 핫스팟 클릭
       → 모든 핫스팟 비활성
       → 클릭한 핫스팟 활성 (빨간 채우기)
       → 상세 패널 데이터 교체 (fadeInUp)
```

### 탭3: AI 퀴즈
```
탭 전환 → 퀴즈 문항 1번 로드
       → 이미지 + 문제 + 4개 선택지 렌더링
       → 선택지 클릭
       → 정답 확인
       → 정답: 해당 선택지 초록 표시
       → 오답: 해당 선택지 빨강 + 정답 선택지 초록 표시
       → 스코어 업데이트 (정답 +1 또는 오답 +1)
       → AI 해설 패널 표시
       → "다음 문제" 버튼 활성화
       → 클릭 → 다음 문항 로드 + 진행바 업데이트
       → 10문항 완료 → 최종 결과 화면 (점수 + 등급)
```

### 최종 결과 등급
| 정답 수 | 등급 | 메시지 |
|---------|------|--------|
| 9~10 | 🏆 정비 마스터 | "현장 투입 가능! 완벽합니다!" |
| 7~8 | 🥈 숙련 정비사 | "거의 다 왔어요! 복습 한 번이면 완벽!" |
| 5~6 | 🔧 견습 정비사 | "기본기가 있네요! 조금 더 공부해봐요." |
| 0~4 | 📚 정비 입문자 | "부품 도감을 다시 살펴보고 도전하세요!" |
