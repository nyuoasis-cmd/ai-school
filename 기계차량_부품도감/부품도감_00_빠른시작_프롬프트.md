자동차 부품 학습 모바일 앱 "파츠피디아(PartsPedia)"를 만들어줘.

## 앱 개요
- 이름: PartsPedia (파츠피디아) — AI 자동차 부품 도감
- 대상: 기계·자동차과 직업계고 학생
- 목적: 자동차 부품을 카드 스와이프로 탐색하고, 인터랙티브 차량 구조도에서 위치를 확인하며, AI 퀴즈로 복습하는 학습 앱
- 스타일: **다크 테마 + 레드 포인트** (#EF4444), 모바일 퍼스트 (390px 기준, 데스크탑에서 폰 프레임 표시)

## 핵심 기능 3탭

### 탭1: 📖 부품 도감
- 카테고리 필터 칩 (전체/엔진계통/제동장치/전기장치/구동·현가)
- **8개 부품 카드를 수평 스와이프**로 탐색 (스크롤 스냅)
- 각 카드: 실사 이미지 + 카테고리 + 한글명 + 영문명 + 설명 + 스펙 4칸 + 난이도 + 상세보기
- 하단 도트 인디케이터 (8개) + "1 / 8 부품" 카운터
- 카테고리 필터 선택 시 해당 카테고리 카드만 표시

### 탭2: 🚗 차량 구조
- 자동차 측면도 SVG + **5개 인터랙티브 핫스팟** (엔진/브레이크/배터리/서스펜션/배기)
- 핫스팟 터치 → 하단 상세 정보 패널 동적 업데이트
- 차량 시스템 분류 카드 4종

### 탭3: 🤖 AI 퀴즈
- 10문항 4지선다 + 진행바 + 정답/오답 스코어
- 정답 선택 후 AI 해설 표시 + "다음 문제" 버튼

## 기술 스택
- React + TypeScript (또는 HTML/CSS/JS)
- Tailwind CSS (다크 테마)
- Claude API (퀴즈 해설 생성, 선택사항)
