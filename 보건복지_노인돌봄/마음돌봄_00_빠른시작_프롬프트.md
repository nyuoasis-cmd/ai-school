# 🚀 MoodCare (마음돌봄) — Replit 빠른 시작 프롬프트

아래 프롬프트를 Replit AI에 **그대로 복붙**하세요.

---

## Replit에 붙여넣을 프롬프트

```
모바일 퍼스트 AI 노인 감정 모니터링 앱을 만들어줘.

[프로젝트 개요]
- 직업계고 보건·복지과 학생을 위한 데모 앱
- 요양시설 어르신들의 감정 상태를 AI로 분석·모니터링하는 서비스
- React (Vite) + Tailwind CSS
- 라이트 테마 (Teal #0D9488 + Lavender #7C3AED 조합)
- 실제 AI API 없이 가상 데이터로 시뮬레이션

[디자인 컨셉]
- 라이트 테마: 배경 #F6F8FA, 카드 #FFFFFF
- 메인 색상: Teal #0D9488 (돌봄/안정감), Lavender #7C3AED (심리/전문성)
- 보조 색상: Green #10B981(기쁨), Blue #3B82F6(슬픔), Amber #F59E0B(불안), Coral #F97066(분노/경고), Purple #7C3AED(외로움)
- 폰트: Archivo Black(타이틀) + JetBrains Mono(수치) + Noto Sans KR(본문)
- 데스크탑: 390x844 폰 프레임 (배경 #E8EDF5, 아이폰 노치 포함)

[탭 구조 - 3개]
1. 🏠 홈 — 대상자 현황 대시보드
2. 🧠 분석 — 대화/관찰 기록 → AI 감정 분석
3. 📈 추이 — 감정 변화 트렌드 + 타임라인

[핵심 기능]
- 6명의 어르신 대상자 카드 (이름·나이·질환·감정상태·스파크라인)
- 한글 성씨 이니셜 아바타 (원형, 고유 색상)
- 대상자 선택 → 관찰 기록 입력 → AI 분석 시작 버튼
- 4단계 로딩 애니메이션 (프로그레스바 25%→50%→75%→100%)
- 분석 결과 바텀시트 팝업 (6각형 레이더 차트 + 감정 점수 + 위험도 + 추천 조치)
- 주간 감정 추이 라인차트 + 최근 분석 타임라인

[데이터]
- 대상자 6명: 김순자(78/치매), 이영호(82/우울증), 박옥순(75/관절염), 정대현(80/당뇨), 한미경(77/고혈압), 최병수(85/파킨슨)
- 감정 6축: 기쁨, 평온, 슬픔, 불안, 분노, 외로움 (각 0~100)
- 위험도 3단계: 🟢안정 / 🟡관찰필요 / 🔴주의
- 빠른 태그 10개: 눈물/식사거름/수면어려움/가족이야기/웃음/산책/TV시청/대화적극/말수줄어듦/짜증

[인터랙션]
- 홈 카드 클릭 → 분석 탭 이동 + 해당 대상자 자동 선택
- 분석 버튼 → 인라인 로딩 (2.8초) → 결과 바텀시트 슬라이드업
- 레이더 차트 중앙에서 펼쳐지는 애니메이션
- 기록 저장 → 토스트 알림 → 바텀시트 닫힘
- 빠른 태그 클릭 시 관찰 기록 텍스트에 자동 추가

[참고 디자인]
- 첨부한 moodcare-design.html 파일의 디자인을 최대한 동일하게 구현해줘
- 아바타는 한글 성씨 이니셜 원형 (사진 대신)
- 히어로 배너는 Unsplash 무료 이미지 사용
```

---

## 사용법

1. Replit에서 **React (Vite)** 프로젝트 생성
2. 위 프롬프트를 Replit AI 채팅에 붙여넣기
3. `moodcare-design.html` 파일도 함께 업로드
4. AI가 생성한 코드 확인 후 추가 수정 요청

## 추가 프롬프트 (필요시)

```
레이더 차트를 SVG로 직접 그려줘. 외부 차트 라이브러리 쓰지 말고.
6각형 그리드 + 데이터 폴리곤 + 축 라벨 + 점수 표시.
중앙에서 펼쳐지는 애니메이션 포함.
```

```
바텀시트 팝업에 backdrop-filter: blur(8px) 배경 처리하고,
슬라이드업 애니메이션은 cubic-bezier(0.32, 0.72, 0, 1) 이징 사용해줘.
```

```
한글 이니셜 아바타 컴포넌트를 만들어줘.
이름의 첫 글자(성씨)를 원형 배경 위에 흰색 텍스트로 표시.
각 대상자마다 고유 색상 지정.
```
