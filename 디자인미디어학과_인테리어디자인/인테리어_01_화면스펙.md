## 탭1: 🔄 비포/애프터

### 공간 선택 칩 (수평 배치)
```
[🛋️ 거실] [🛏️ 침실] [🍳 주방] [🛁 욕실]
```
- 칩: padding 7px 16px, border-radius 20px, font-size 12px
- 활성: 핑크 그라디언트 배경 + 흰 글씨 + glow shadow
- 비활성: 흰 배경 + #6B5460 글씨 + 보더
- 칩 클릭 → 해당 공간 카드 표시 + 도트/카운터 업데이트

### 드래그 힌트
- "↔ 슬라이더를 드래그하여 비교하세요"
- font-size 11px, 펄스 애니메이션

### 비포/애프터 카드 (4개, 칩으로 전환)
```
┌──────────────────────────────┐
│ ┌─────────┬──┬──────────┐    │
│ │         │▓▓│          │    │
│ │ BEFORE  │◂▸│  AFTER ✨│    │  ← 260px 높이
│ │         │▓▓│          │    │
│ └─────────┴──┴──────────┘    │
├──────────────────────────────┤
│ 모던 미니멀 거실              │  ← Playfair 18px bold
│ Modern Minimal Style         │  ← 핑크 12px
│                              │
│ 설명 텍스트 2줄...            │  ← 13px secondary
│                              │
│ [🪑 가구교체] [💡 조명추가]    │
│ [🎨 벽컬러변경] [🌿 식물배치]  │  ← 변경 포인트 태그
└──────────────────────────────┘
```

### 비포/애프터 슬라이더 구조
- Before 이미지: position absolute, 전체 영역
- After 이미지: position absolute, **clip-path: inset(0 50% 0 0)** (왼쪽 절반만 보임)
- 수직 구분선: width 3px, 흰색, box-shadow
- 핸들: 44px 원형, 흰 배경 + 핑크 보더 3px + 그림자, "◂ ▸" 화살표
- 드래그/터치: clientX로 비율 계산 → clip-path + 구분선 + 핸들 위치 실시간 업데이트
- 범위: 5% ~ 95%

### BEFORE/AFTER 라벨
- BEFORE: 좌상단, 검정 반투명 배경, 흰 글씨
- AFTER ✨: 우상단, 핑크 반투명 배경, 흰 글씨

### 변경 포인트 태그
- padding 4px 10px, border-radius 16px
- 핑크 소프트 배경 + 핑크 텍스트 + 연한 보더

### 도트 인디케이터 + 카운터
- 4개 도트, 활성: 20px 핑크 바
- "1 / 4 공간" (핑크 강조)

## 인터랙션 정리

| 요소 | 동작 | 효과 |
|------|------|------|
| 탭 전환 | 클릭 | 컨텐츠 교체 + 활성 인디케이터 |
| 공간 칩 | 클릭 | 해당 공간 카드 표시 + 슬라이더 50% 리셋 + 도트/카운터 업데이트 |
| BA 슬라이더 | 드래그/터치 | clip-path + 구분선 + 핸들 실시간 이동 |
| 스타일 카드 | 클릭 | scale(0.98) 피드백 → 상세 페이지 이동 |
| AI 옵션 | 클릭 | 그룹 내 단일 선택 토글 |
| AI 추천 버튼 | 클릭 | AI 결과 생성/표시 |
