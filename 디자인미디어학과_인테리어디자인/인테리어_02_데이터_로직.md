## 5. 로직 흐름

### 탭1: 비포/애프터
```
앱 로드 → 거실 카드 기본 표시 (슬라이더 50%)
       → 공간 칩 클릭
       → 모든 카드 숨김
       → 선택한 공간 카드 표시
       → 슬라이더 50% 리셋
       → 도트/카운터 업데이트
       
       → 슬라이더 드래그/터치
       → mousedown/touchstart → isDragging = true
       → mousemove/touchmove → clientX 기준 비율 계산 (5~95%)
       → After 이미지 clip-path 업데이트
       → 구분선 left 업데이트
       → 핸들 left 업데이트
       → mouseup/touchend → isDragging = false
```

### 탭2: 스타일 갤러리
```
탭 전환 → 4개 스타일 카드 렌더링 (fadeInUp 순차 애니메이션)
       → 카드 클릭 → 상세 페이지 이동 (선택사항)
```

### 탭3: AI 추천
```
탭 전환 → 프롬프트 카드 표시
       → 공간 선택 (그룹 내 단일 선택)
       → 분위기 선택 (그룹 내 단일 선택)
       → 예산 선택 (그룹 내 단일 선택)
       → "AI 추천 받기" 클릭
       → 선택 조합으로 프리셋 검색
       → 있으면: 즉시 결과 카드 표시
       → 없으면: Claude API 호출 또는 유사 프리셋 표시
       → 결과 카드: 이미지 + 타이틀 + 설명 + 팔레트 + 팁
```
