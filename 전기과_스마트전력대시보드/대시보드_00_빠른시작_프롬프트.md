# 🚀 스마트 전력 모니터링 대시보드 — Replit 빠른 시작 프롬프트

아래 프롬프트를 Replit AI에 **그대로 복붙**하세요.

## Replit에 붙여넣을 프롬프트

```
모바일 퍼스트 스마트 전력 모니터링 대시보드 앱을 만들어줘.

[프로젝트 개요]
- 직업계고 전기·전자과 학생을 위한 데모 앱
- 학교 구역별 전력 사용량을 실시간 시뮬레이션하는 대시보드
- React (Vite) + Tailwind CSS
- 다크 테마, 사이언/일렉트릭 블루 색상
- 가상 데이터가 2초마다 미세하게 변동하는 실시간 느낌

[화면 구조 - 3탭]
상단 헤더 → 탭 바 (대시보드 / 구역별 / 알림) → 콘텐츠 영역 (스크롤)

[헤더]
- 좌: ⚡ 아이콘 (사이언 배경)
- 중: "스마트 전력 모니터링" + DEMO 배지 + "한국전기전자고등학교"
- 우: "LIVE" 배지 (초록 점 깜빡임)

[탭 바]
- 📊 대시보드 | 🏫 구역별 | 🔔 알림
- 활성 탭: 사이언 색 + 하단 보더

[탭1: 대시보드]
1. 요약 카드 3개 (가로 3열)
   - 현재 전력: 47.2 kW (사이언) ▼3.1%
   - 오늘 누적: 312 kWh (블루) ▼5.2%
   - 이번달 요금: 847 만원 (옐로) ▲2.3%

2. 실시간 전력 게이지 (반원형 SVG)
   - 0~100% 아크, 사이언 색 + glow 효과
   - 중앙: 큰 숫자 (68%) + "47.2 / 70 kW"
   - 하단: 상태 배지 (정상/주의/위험 → 색상 자동 변경)
   - 최대 허용: 70 kW | 계약 전력: 100 kW

3. 시간대별 바 차트 (06시~17시, 12칸)
   - 막대 높이 = 사용량 비례
   - 색상: 낮음(초록) / 보통(사이언) / 높음(오렌지) / 피크(빨강)
   - 상단에 수치 표시

4. 실시간 수치 카드 4개 (2x2 그리드)
   - 전압: 221V (사이언) / 정상 범위 220±10V
   - 전류: 214A (초록) / 정격 300A
   - 주파수: 60.0Hz (퍼플) / 정상 59.9~60.1
   - 역률: 96.5% (옐로) / 목표 95% 이상

5. AI 절전 분석 카드
   - 🤖 아이콘 + "AI 절전 분석" 라벨 (사이언)
   - 분석 텍스트 (3층 컴퓨터실 34% 점유, 월 12만원 절감 가능)

[탭2: 구역별]
1. 구역 카드 리스트 (6개, 세로 1열)
   - 아이콘 + 구역명 + 점유율 + 프로그레스바 + 전력값 + 상태
   - 구역: 💻컴퓨터실, 🔧전기실습실, 📚교실동, 🏀체육관, 🍚급식실, 🏢행정동

2. 구역별 비율 수평 바 차트
   - 구역명 + 바 + 퍼센트

3. 절전 등급 카드
   - 🅱 양호 등급, A등급까지 8% 추가 절감 필요

[탭3: 알림]
- 알림 카드 리스트 (좌측 컬러 보더)
  - 🚨 위험(빨강): 순간 최대전력 초과
  - ⚠️ 경고(옐로): 역률 저하
  - 💡 정보(사이언): 자동 절전 실행
  - 📊 정보: 일일 리포트
  - 🌡️ 경고: 변압기 온도 상승
  - ✅ 정보: 월간 절전 목표

[실시간 시뮬레이션]
- 2초마다 setInterval로 수치 변동
- 전력: 47.2 ± 2 kW 범위
- 전압: 221 ± 2 V
- 전류: 214 ± 5 A
- 주파수: 60.0 ± 0.1 Hz
- 역률: 96.5 ± 0.75 %
- 게이지 퍼센트 자동 계산 (전력/70*100)
- 게이지 색상: <75% 사이언(정상), 75~90% 옐로(주의), 90%+ 빨강(위험)

[디자인 - 다크 테마]
- 배경: #0B1121 (메인), #111B33 (헤더), #162040 (카드)
- 텍스트: #E8ECF4 (메인), #8B9BC0 (보조), #4E6190 (비활성)
- 액센트: #00D4FF (사이언), #00E676 (초록), #FFD600 (옐로), #FF9100 (오렌지), #FF5252 (빨강), #448AFF (블루), #B388FF (퍼플)
- 카드 border: 1px solid #243056
- 각 색상별 dim 버전: rgba(색상, 0.15)로 배경에 사용
- 폰트: Noto Sans KR
- 데스크탑: 390x844 폰 프레임 (배경 #0a0e1a)

[참고]
- 첨부된 power-dashboard-design.html이 완성된 UI 디자인 레퍼런스
- 이 HTML과 동일한 룩앤필로 React 컴포넌트 구현
- AI API 연동 없음. 모든 데이터는 하드코딩 + setInterval 시뮬레이션
```

## 파일 전달 체크리스트

| 파일 | 용도 | 필수 |
|------|------|------|
| `power-dashboard-design.html` | UI 디자인 레퍼런스 | ✅ |
| `대시보드_00_빠른시작_프롬프트.md` | 이 파일 (복붙용) | ✅ |
| `대시보드_01_화면_스펙.md` | 화면별 상세 레이아웃 | ✅ |
| `대시보드_02_데이터_시뮬레이션.md` | 구역/알림 데이터 + 시뮬레이션 로직 | ✅ |

## 진행 순서

1. React (Vite) 프로젝트 생성
2. 위 프롬프트 + HTML 레퍼런스 파일 첨부
3. 다크 테마 + 레이아웃 (헤더, 탭, 콘텐츠) 먼저
4. 대시보드 탭: 요약 카드 → 게이지 → 바 차트 → 상태 카드
5. 구역별 탭: 구역 리스트 → 비율 차트 → 등급 카드
6. 알림 탭: 알림 리스트
7. setInterval 실시간 시뮬레이션 적용
8. 데스크탑 폰 프레임
