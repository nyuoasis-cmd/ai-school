## 3. 탭1: 대시보드

### 3-1. 요약 카드 (3열 그리드)

```
┌──────────┬──────────┬──────────┐
│ 현재 전력  │ 오늘 누적  │ 이번달 요금│
│  47.2    │   312    │   847   │
│   kW     │   kWh    │   만원   │
│ ▼3.1%    │ ▼5.2%    │ ▲2.3%   │
└──────────┴──────────┴──────────┘
```

- 카드: surface bg, border, radius 12px, padding 12px, text-center
- 라벨: 10px, text2
- 값: 20px, 800weight, 각각 다른 색상 (cyan/blue/yellow)
- 단위: 11px, text2
- 변화: 10px, 600weight, 감소=green, 증가=red

### 3-2. 실시간 전력 게이지

```
┌──────────────────────────────────┐
│      (radial cyan glow bg)        │
│    ⚡ 실시간 전력 사용률             │
│                                   │
│       ╭────────────╮             │
│     ╱                ╲           │
│    │                  │          │
│    ─────────────────────         │
│            68                    │
│       47.2 / 70 kW              │
│                                   │
│       [✅ 정상 운영 중]            │
│   최대 허용: 70kW | 계약: 100kW   │
└──────────────────────────────────┘
```

- SVG 반원 게이지: viewBox 200x120
  - 배경 arc: stroke border, stroke-width 10
  - 채움 arc: stroke cyan, stroke-width 10, stroke-linecap round
  - stroke-dasharray: 251 (반원 둘레)
  - stroke-dashoffset: 251 - (251 × pct / 100)
  - filter: `drop-shadow(0 0 6px cyan-glow)`
- 중앙 값: 36px, 900weight, cyan
- kWh: 12px, text2
- 배경 글로우: `radial-gradient(circle at 50% 80%, cyan-dim 0%, transparent 50%)`
- 상태 배지:
  - ~74%: 정상(good) → green-dim bg, green text, ✅
  - 75~89%: 높음(warn) → yellow-dim bg, yellow text, ⚠️
  - 90%~: 주의(danger) → red-dim bg, red text, 🚨
- 게이지 stroke 색상도 상태에 따라 cyan → yellow → red 변경

### 3-3. 시간대별 바 차트


- 12개 바 (06~17시)

- 색상 등급:

  - 30~44: cyan (mid)
  - 45~59: orange (high)
  - 60~: red (peak)
- 상단: 8px 값 표시, 하단: 9px 시간 라벨

## 7. 애니메이션

| 이름 | 용도 | 상세 |
|------|------|------|
| fadeUp | 카드 등장 | opacity 0→1, translateY 10→0, 0.4s |
| blink | LIVE 점 | opacity 1→0.3→1, 1.5s infinite |
| gauge-fill | 게이지 변동 | stroke-dashoffset transition 1s ease |
| glow | 게이지 글로우 | box-shadow 사이언 dim↔glow, pulse |
| bar-grow | 바 차트 등장 | height 0→목표, 0.8s ease |
