## 5. 타이핑 인디케이터

```
[🐱] [ ●  ●  ● ]
```

- 아바타: 30x30 (메시지 아바타와 동일)
- 버블: 흰 배경, radius 18px 18px 18px 4px
- 점: 7px 원, `#9398B0`
- 애니메이션: 순차 바운스 (0 → 0.2s → 0.4s 딜레이)
  - 30% 시점: opacity 1, translateY(-4px)
  - 0%/60%/100%: opacity 0.3, translateY(0)

## 10. 응답 타이밍 시퀀스

```
버튼 클릭/입력
  → 0ms: 유저 메시지 표시 + 웰컴/버튼 숨김
  → 0ms: 타이핑 인디케이터 표시
  → 600ms: 텍스트 응답 표시 (있으면) + 타이핑 다시
  → 1200ms: 카드 응답 표시 (있으면) + 타이핑 다시
  → 1800ms: 후속 멘트 표시
  → 자동 스크롤 (각 단계마다)
```
