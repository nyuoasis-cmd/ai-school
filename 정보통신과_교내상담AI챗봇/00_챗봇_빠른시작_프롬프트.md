# 🚀 교내 AI 상담 도우미 — Replit 빠른 시작 프롬프트

아래 프롬프트를 Replit AI에 **그대로 복붙**하세요.

---

## Replit에 붙여넣을 프롬프트

```
모바일 퍼스트 교내 AI 상담 도우미 챗봇 앱을 만들어줘.

[프로젝트 개요]
- 직업계고 정보·통신 전공 학생을 위한 데모 앱
- 학교 정보(급식, 시간표, 학사일정 등)를 AI가 답변하는 챗봇
- React (Vite) + Tailwind CSS
- OpenAI Chat API로 답변 생성

[캐릭터 설정]
- 이름: "또리" (학교 정보통 고양이 🐱)
- 말투: "~다냥", "알려줄게냥!", "화이팅이다냥 🐾"
- 성격: 친근하고 똑똑한 학교 정보통

[화면 구조 - 단일 채팅 화면]
위에서 아래로 순서대로:

1. 헤더 (sticky top)
   - 🐱 아바타 (퍼플 그라데이션 원형) + "또리" + DEMO 배지 + "응답 가능" 상태
   - 우측: "정보·통신" 태그

2. 웰컴 카드 (상단 고정, 퍼플 그라데이션 배경)
   - 🐱 이모지 + "안녕, 나는 또리다냥!" + 설명

3. 자주 묻는 질문 (1열 세로 리스트, 6개)
   - 각 버튼: [아이콘] + 질문 텍스트 (가로 배치)
   - 🍚 오늘 급식 뭐야?
   - 📅 이번 주 시간표 알려줘
   - 📋 학사일정 알려줘
   - 🏫 교무실 어디야?
   - 📶 와이파이 비번 뭐야?
   - 🎮 동아리 정보 알려줘

4. 채팅 메시지 영역 (대화 시작하면 여기에 쌓임)

5. 입력창 (sticky bottom)
   - placeholder: "또리에게 물어봐냥~ 🐾"
   - 전송 버튼: 🐾 (퍼플 원형)

[버튼 클릭 시 흐름]
버튼 탭 → 웰컴카드+버튼 숨김 → 유저 메시지 표시 → 타이핑 인디케이터(점3개 바운스) → 봇 텍스트 응답 → 카드 응답 → 후속 멘트

[카드형 답변 3종]

1. 🍚 급식 카드
   - 헤더: 오렌지 그라데이션, "오늘의 급식" + 날짜
   - 본문: 메뉴 리스트 (이모지 + 음식명 + 칼로리)
   - 푸터: 오렌지 배경, 총 칼로리

2. 📅 시간표 카드
   - 헤더: 블루 그라데이션, "이번 주 시간표" + 반
   - 본문: 요일x교시 테이블, 오늘 열 하이라이트 (퍼플)

3. 📋 학사일정 카드
   - 헤더: 핑크 그라데이션, "학사일정" + 월
   - 본문: 날짜 박스 + 일정명 + 설명 + 태그(시험/행사/방학)

[텍스트 답변 - 카드 없이 텍스트만]
- 교내 위치: 주요 장소 목록 (교무실, 행정실, 진로상담실 등)
- 와이파이: SSID + 비밀번호
- 동아리: 동아리 3개 소개 (활동, 시간, 장소)

[AI API 연동 - 직접 입력 시]
- 키워드 매칭으로 분류: 급식/시간표/일정/위치/와이파이/동아리
- 매칭 안 되면 OpenAI API로 일반 답변 생성
- 시스템 프롬프트: "너는 학교 AI 도우미 '또리'야. 고양이 캐릭터로 ~다냥 말투를 써. 학교 정보를 친근하게 알려줘."

[디자인]
- 색상: bg #F0F2F8, accent #6C5CE7(퍼플), blue #3B82F6, green #10B981, orange #F59E0B, pink #EC4899
- 폰트: Noto Sans KR
- 카드: radius 12~16px, shadow 미세하게
- 봇 메시지: 흰 배경, 좌측 정렬, 좌하단 radius 작게
- 유저 메시지: 퍼플 배경 흰 글씨, 우측 정렬, 우하단 radius 작게
- 타이핑: 점 3개 바운스 애니메이션
- 데스크탑: 390x844 폰 프레임 (배경 #DDD8EE)
- 모바일: 풀스크린

[참고]
- 첨부된 school-chatbot-design.html이 완성된 UI 디자인 레퍼런스
- 이 HTML과 동일한 룩앤필로 React 컴포넌트 구현
```

## 파일 전달 체크리스트

| 파일 | 용도 | 필수 |
|------|------|------|
| `school-chatbot-design.html` | UI 디자인 레퍼런스 | ✅ |
| `00_챗봇_빠른시작_프롬프트.md` | 이 파일 (복붙용 프롬프트) | ✅ |
| `01_챗봇_화면_스펙.md` | 화면별 상세 레이아웃 | ✅ |
| `02_챗봇_데이터_API.md` | 응답 데이터 + API 연동 | ✅ |

## 진행 순서

1. React (Vite) 프로젝트 생성
2. 위 프롬프트 + HTML 레퍼런스 파일 첨부
3. 하드코딩 응답(샘플 6종) 먼저 동작 확인
4. OpenAI API 키 설정 → 자유 입력 AI 답변 추가
5. 타이핑 애니메이션, 카드 등장 순서 조정
6. 데스크탑 폰 프레임 적용
