## 작업 규칙 (이전 세션에서 확립)
1. **각 앱마다 UI 패턴 다르게** (카메라, 채팅, 대시보드, 슬라이더 등)
2. **색상 테마 다르게** (오렌지, 퍼플, 사이언, 에메랄드 등)
3. **라이트/다크 교차** (3개 라이트 + 1개 다크 현재)
4. **산출물 3종**: HTML 프로토타입 + Replit 빠른시작 프롬프트 + 화면 스펙 + 로직 문서
5. **모바일 퍼스트**: 390px 기준, 데스크탑에서 폰 프레임
6. **학과 특성 반영**: 해당 분야 실무와 연결되는 시뮬레이션/도구
7. **상품 사진은 실사** (Unsplash 등 활용)
