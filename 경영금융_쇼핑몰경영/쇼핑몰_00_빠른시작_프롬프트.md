모바일 퍼스트 모의 창업 쇼핑몰 시뮬레이터 앱을 만들어줘.

- 직업계고 경영·금융과 학생을 위한 모의 창업 판매 관리 시뮬레이션 앱

- 학생이 가격과 할인율을 조절하면 판매량·매출·이익이 실시간 변동

- 라이트 테마, 에메랄드 그린(#10B981) 기반 색상
- 상품 썸네일은 실제 사진 (Unsplash URL 사용)

- 헤더 그라디언트: #10B981 → #059669
- 상품1 포인트: #3B82F6 (블루) / 배경: #EBF2FF
- 상품2 포인트: #8B5CF6 (퍼플) / 배경: #EDE9FE
- 수익/긍정: #10B981, 손실/경고: #EF4444, 할인: #F59E0B

[화면 구조 - 3탭]
상단 헤더(그린 그라디언트) → 탭 바 → 콘텐츠 영역 (스크롤)

[헤더]
- 좌: 🛍️ 아이콘 + "모의 창업 쇼핑몰" + DEMO 배지
- 우: ↺ 초기화 버튼
- 하단: 요약 스탯 4칸 (투명 칩)
  - 총 매출 ___만원/월 | 순이익 ___만원/월 | 이익률 ___% | 총 판매 ___개/월
  - 슬라이더 조절 시 실시간 업데이트

[탭 바]
- 🏪 내 쇼핑몰 | 📊 판매 분석 | 🤖 AI 전략
- 활성 탭: 에메랄드 그린 + 하단 보더

[탭1: 내 쇼핑몰] — 상품 2개만 표시
각 상품은 카드 형태로:

상품 1: 오버핏 맨투맨
- 사진: https://images.unsplash.com/photo-1556821840-3a63f95609a7?w=200&h=200&fit=crop&q=80

- 원가: 12,000원, 기본가: 25,000원, 기본판매량: 120개/월
- 포인트 색상: #3B82F6 (블루)

상품 2: 와이드 데님 팬츠
- 사진: https://images.unsplash.com/photo-1542272604-787c3835535d?w=200&h=200&fit=crop&q=80
- 설명: 스트릿 감성 · S / M / L
- 원가: 15,000원, 기본가: 35,000원, 기본판매량: 80개/월
- 포인트 색상: #8B5CF6 (퍼플)

카드 구조 (각 상품):
┌─────────────────────────────┐
│ [사진 82x82]  상품명           │
│              설명              │
│              [원가 태그] [기본 태그] [할인% 태그] │
├─────────────────────────────┤
│ 💵 판매 가격         ___원    │
│ [━━━━━━━━━●━━━] 슬라이더     │
│ 🏷️ 할인율            ___%    │
│ [━━━━━━━━●━━━━] 슬라이더     │

- 판매가 슬라이더: 원가 ~ 기본가×2 범위, 500원 단위
- 할인율 슬라이더: 0% ~ 50% 범위, 5% 단위
- 슬라이더 조절 시 하단 결과 칩 3개 실시간 업데이트
- 이익이 적자면 빨간색으로 표시

[탭2: 판매 분석]
1. KPI 카드 4개 (2x2 그리드)
   - 💰 월 매출 ___만 (▲/▼ __% 기본대비)
   - 📦 총 판매 ___개 (▲/▼ __%)
   - 🏦 순이익 ___만 (▲/▼ __%)
   - 📊 이익률 ___% (양호/개선)

2. 상품별 매출 비교 (수평 바 차트)
   - 오버핏 맨투맨: 블루 바 + 금액 표시
   - 와이드 데님 팬츠: 퍼플 바 + 금액 표시

3. 이익률 비교 (수평 바 차트)
   - 40%+ 초록, 20~40% 오렌지, 20%미만 빨강

4. 가격 변경 효과 카드
   - 기본 가격 → 현재 설정 비교
   - 매출 변화율, 이익 변화율 표시

[탭3: AI 전략]
1. 경영 전략 점수 카드 (그린 그라디언트 배경)
   - 🏆 이모지 + 점수(0~100) + 등급
   - 등급: ⭐초보(~40), 🌱배우는중(40~55), 📈성장중(55~70), 💎유능한CEO(70~85), 🏆경영천재(85~)
   - 하위 점수: 가격전략 / 수익성 / 판매전략

2. AI 경영 분석 카드 (민트 그라디언트)
   - 🤖 아이콘 + "AI 경영 분석"
   - 동적 텍스트: 매출 비중, 적자 경고, 할인율 분석, 이익률 평가

3. 추천 전략 3개 카드
   - ①(초록) 고마진 유지 전략
   - ②(블루) 볼륨 확대 전략
   - ③(오렌지) 차별 가격 전략

[가격-수요 시뮬레이션 모델]
실판매가 = 판매가 × (1 - 할인율/100)
가격비율 = 기본가 / 실판매가
할인부스트 = 할인>0 ? 1 + (할인/100)×0.8 : 1
예상판매 = 기본판매 × (가격비율)^1.5 × 할인부스트
매출 = 실판매가 × 예상판매
이익 = (실판매가 - 원가) × 예상판매
이익률 = 이익 / 매출 × 100

[핵심 UX]
- 슬라이더 조절 즉시 → 상품 카드 결과, 헤더 총합, 분석 탭, AI 탭 모두 연동
- 초기화 버튼 누르면 모든 가격/할인이 기본값으로 복원
- 탭 전환 시 최신 데이터로 차트/분석 다시 렌더링
- 모바일 390px 기준, 데스크탑에서는 폰 프레임으로 표시

[디자인 주의사항]
- 라이트 테마 (어두운 배경 아님)
- 상품 사진은 정사각형 82x82px, border-radius:12px
- 카드는 둥근 모서리(16px) + 미세 테두리 + 약한 그림자
- 슬라이더 thumb: 20px 원형, 메인 색상
- 결과 칩: 각 상품 색상 배경
- 전체적으로 깔끔하고 감성적인 쇼핑몰 느낌
