# 비즈쿨 AI 창업교육 커리큘럼 페이지 — CLI 구현 지시문서

## 📌 프로젝트 개요

비즈쿨 AI 창업교육 12차시 커리큘럼을 보여주는 인터랙티브 React 페이지입니다.
3개 블록(4/8/12차시)이 카드 형태로 표시되고, 클릭하면 모달로 상세 차시 정보를 확인할 수 있습니다.

---

## 📁 파일 구조

```
src/
  components/
    BizcoolCurriculum.jsx   ← 메인 컴포넌트 (첨부된 bizcool-curriculum.jsx)
```

이 파일은 **단일 파일 컴포넌트**입니다. 별도 CSS 파일 없이 inline style + useEffect로 글로벌 스타일을 주입합니다.

---

## 🏗️ 아키텍처

### 컴포넌트 구성 (4개)

```
BizcoolCurriculum (메인)
  └─ CourseCard (블록 카드) × 3
       └─ Modal (상세 모달)
            └─ ClassCard (차시 카드) × 2 per visit
```

### 데이터 구조 (`const C` 배열)

```javascript
// 블록 레벨
{
  id: "s4",                    // 고유 ID
  tier: "스타터",               // 블록 이름 (스타터/빌더/프로)
  hours: "4차시",              // 총 차시
  meta: "50분 × 4교시 · 2회 방문",  // 메타 정보
  visits: ["1회차","2회차"],    // 방문 회차 (탭으로 사용)
  title: "AI 창업 아이디어 스프린트",  // 블록 제목
  desc: "...",                 // 블록 설명
  tags: ["시장세분화","AI 리서치","BMC","SCAMPER"],  // 태그
  hl: {icon:"🔥", text:"..."},  // 하이라이트 배지
  col: "#00d68f",              // 블록 테마 색상
  dim: "#00d68f15",            // 블록 테마 색상 (투명)
  prereq: null,                // 선수 과정 (null / "4차시 포함" / "8차시 포함")
  grad: "🎓 아이디어 제안서 + 1분 피칭 영상",  // 졸업 산출물
  flow: "발견하고 → 고객 찾고 → ...",  // 흐름 요약
  detail: [...]                // 방문별 상세 (아래 참조)
}

// 방문 레벨 (detail 배열 내)
{
  visit: "1회차",              // 탭 라벨
  vt: "아이디어 발굴 & AI 시장분석",  // 방문 제목
  classes: [...]               // 차시 배열 (아래 참조)
}

// 차시 레벨 (classes 배열 내)
{
  n: "1",                      // 차시 번호
  t: "창업 마인드셋 & 시장세분화",  // 차시 제목
  step: "STEP 01",             // 스텝 라벨
  ref: "스타트업 바이블 1단계",    // 참조 교재
  min: 50,                     // 수업 시간 (분)
  type: "이론+실습",            // 수업 유형 (이론+실습 / 실습 / 발표)
  desc: "...",                 // 차시 설명
  act: ["활동1","활동2","활동3"],  // 핵심 활동 (3개)
  tools: ["ChatGPT","Perplexity"],  // 사용 도구
  out: "시장 기회 매트릭스"      // 산출물
}
```

|------|-----|

| 모달 배경 | `#111119` |
| 차시카드 배경 | `#1a1a28` |
| 텍스트 (제목) | `#e8e8f0` |
| 텍스트 (본문) | `#888` ~ `#999` |
| 텍스트 (서브) | `#555` ~ `#666` |
| Block 1 (스타터) | `#00d68f` / `#00d68f15` |
| Block 2 (빌더) | `#4e8cff` / `#4e8cff15` |
| Block 3 (프로) | `#a855f7` / `#a855f715` |
| 브랜드 액센트 | `#ff6b2b` |

### 폰트

```
본문: 'Noto Sans KR', sans-serif
모노: 'JetBrains Mono', monospace  (라벨, 메타, 태그 등)
```

Google Fonts import를 useEffect에서 동적으로 주입합니다.

### 수업 유형 배지

```javascript
const TC = {
  "이론+실습": {bg:"#4e8cff18", c:"#4e8cff", i:"📖"},
  "실습":      {bg:"#00d68f18", c:"#00d68f", i:"⚡"},
  "발표":      {bg:"#fbbf2418", c:"#fbbf24", i:"🎤"},
};
```

### 애니메이션

```css
cdIn: 카드 등장 (translateY 32px → 0, stagger 0.12s)
mbIn: 모달 배경 페이드인
mdIn: 모달 콘텐츠 스케일+슬라이드
```

## 🔧 CLI로 수정 시 참고

### 차시 내용 수정
`const C` 배열 내 해당 차시의 데이터를 직접 수정하면 됩니다.
각 차시는 `{n, t, step, ref, min, type, desc, act, tools, out}` 구조입니다.

### 블록 추가/삭제
`const C` 배열에 블록 객체를 추가/제거하면 카드와 프로그레스바가 자동 반영됩니다.

### 색상 변경
각 블록의 `col`(테마 색상)과 `dim`(투명 배경)을 변경합니다.
`dim`은 `col` + `15` (hex opacity)로 설정합니다.

### 새로운 수업 유형 추가
`TC` 객체에 `{bg, c, i}` 형태로 추가합니다.
