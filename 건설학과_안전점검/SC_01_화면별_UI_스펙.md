## 4. 점검 탭 — 체크리스트 위저드

구역 선택 시 구역 선택 화면 숨기고 위저드 패널 표시.

### 4-1. 구역 이미지 헤더

```
┌─────────────────────────────────┐
│ [←]                             │
│                                 │
│  [선택 구역 실사 이미지 120px]   │
│                                 │
│  굴착 현장        [고위험]       │
└─────────────────────────────────┘
```

- 높이: 120px, 선택 구역의 Unsplash 이미지 사용
- 오버레이: gradient (transparent → 70% dark)
- 뒤로가기 버튼: 절대 위치 top-left, 36px 원형, rgba(0,0,0,0.4), backdrop-blur
- 구역명: Archivo Black 18px, 흰색
- 위험 배지: 10px, 둥근 pill, 빨간/주황 배경

### 4-2. 스텝 프로그레스

```
  ①───②───③───④     3 / 4 단계
```

- 원형 스텝 닷 (28×28px, JetBrains Mono 12px)
  - 대기: bg2 배경, border, 회색 텍스트
  - active: 앰버 배경, 흰 텍스트, 앰버 글로우 (box-shadow 4px)
  - done: 그린 배경, 흰색 "✓"
- 연결선: height 3px, 대기=border색, done=그린
- 카운터: JetBrains Mono 11px, 우측 정렬

### 4-3. 카테고리 헤더

```

│            탭: ✅ 적합 · 🚩 위반  │

### 4-4. 체크리스트 항목

```
┌─────────────────────────────────┐
│ [☐] 안전모 착용 여부         [🚩]│
└─────────────────────────────────┘
┌─────────────────────────────────┐  ← 적합 상태
│ [✓] 안전화 착용 상태         [🚩]│  (그린 테두리, 그린 배경)
└─────────────────────────────────┘
┌─────────────────────────────────┐  ← 위반 상태
│ [!] 보호안경 착용 확인       [🚩]│  (레드 테두리, 레드 배경)
└─────────────────────────────────┘
```

- 항목 카드: flex row, padding 14px 16px, radius 12px
- 체크박스(24×24px): 미선택=border만, 적합=그린fill+✓, 위반=레드fill+!
- 텍스트: 14px, weight 500
  - 적합 시: 그린 색상
  - 위반 시: 레드 색상
- 🚩 버튼: 28×28px, 기본 opacity 0.3, 위반 시 opacity 1
- **인터랙션**:
  - 카드 영역 탭 → 적합 토글 (미선택↔적합)
  - 🚩 버튼 탭 → 위반 토글 (미선택↔위반)
  - event.stopPropagation()으로 버블링 방지

- 이전 버튼: surface2 배경, text2 색, border

### 5-1. 오버레이

- 배경: rgba(42,32,16,0.6), backdrop-filter blur(8px)
- 시트: max-height 92vh, radius 24px top, surface 배경
- 진입 애니메이션: translateY(100%) → 0 (0.4s cubic-bezier)
- 핸들바: 40×4px, border색, 상단 중앙

### 5-2. 구역 이미지 헤더 (120px)

- 해당 구역 Unsplash 이미지, brightness(0.45)
- 중앙 정렬: 구역명 + 등급 텍스트 (흰색)

### 5-3. 점수 링

```
        ╭──────╮
       │  85   │    ← Archivo Black 40px
       │ / 100 │    ← JetBrains Mono 12px
        ╰──────╯
      🟢 안전           ← 등급
  현장 안전 상태 양호    ← 코멘트
```

- SVG 원형 (140×140px, r=60, stroke-width=10)
- 배경 링: bg2 색
- 채움 링: strokeDasharray=377, offset 애니메이션 (1s ease)
  - ≥80: 그린, ≥50: 앰버, <50: 레드
- 등급 텍스트: 18px weight 800, 색상은 점수에 따라
- 코멘트: 13px, text2

### 5-4. 요약 카드

```
┌──────────┬──────────┬──────────┐
│    ✅    │    🚩    │    ⬜    │
