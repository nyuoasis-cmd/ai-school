# 🚀 SafeCheck 건설현장 안전점검 — Replit 빠른 시작 프롬프트

아래 프롬프트를 Replit AI에 **그대로 복붙**하세요.
상세 스펙은 다른 문서들을 참고하면 됩니다.

---

## Replit에 붙여넣을 프롬프트

```
모바일 퍼스트 건설현장 안전점검 앱을 만들어줘.

[프로젝트 개요]
- 직업계고 건설과 학생을 위한 데모 앱
- 건설현장 구역별 안전 체크리스트를 단계별로 점검하고 점수를 산출하는 앱
- React (Vite) + Tailwind CSS
- AI 연동 없이 프론트엔드 로직으로만 동작
- 모든 이미지는 Unsplash 실사 건설 현장 사진 사용 (이모지 아이콘 X)

[디자인 컨셉]
- 라이트 테마, 앰버/옐로 톤 (안전모·경고 색상)
- 배경: #FFFCF5, 카드: #FFFFFF, 보조배경: #FFF7E6
- 메인 색상: 앰버 #E8960C
- 안전(적합): 그린 #1B9E3E, 위반: 레드 #DC3545, 주의: 오렌지 #E86B0C
- 폰트: Archivo Black (타이틀) + JetBrains Mono (수치) + Noto Sans KR (본문)
- 카드: radius 16px, border 1.5px solid #F0E4CC
- 데스크탑: 390x844 폰 프레임 (다크 배경, 노치, radius 44px)

[탭 구조 - 2개]
1. ✅ 점검 — 구역 선택 → 단계별 체크리스트 → 결과
2. 📋 기록 — 저장된 점검 히스토리

[✅ 점검 탭 — 구역 선택 화면]
1. 히어로 배너 (180px, Unsplash 건설현장 전경 사진 + 그라데이션 오버레이)
   - "건설현장 안전점검" 타이틀 + "점검할 현장 구역을 선택하세요"
2. 구역 카드 리스트 (6개, 세로 배치)
   - 각 카드: [구역 실사 사진 100px] + [구역명 + 설명 + 항목 수 + 위험등급 배지]
   - 구역 6개:
     - 굴착 현장: 토공사·터파기 작업 구역, 16개 항목, 고위험
     - 철골 작업: 철골 조립·용접 작업 구역, 14개 항목, 고위험
     - 비계 작업: 외부 비계 설치·해체 구역, 15개 항목, 고위험
     - 고소 작업: 2m 이상 높이 작업 구역, 14개 항목, 고위험
     - 전기 작업: 전기 배선·배전 작업 구역, 12개 항목, 중위험
     - 크레인 작업: 양중기·크레인 운반 구역, 13개 항목, 중위험

[✅ 점검 탭 — 체크리스트 위저드]
구역 선택 시 4단계 스텝 위저드 진행:

1. 상단: 선택 구역 실사 이미지 배너 (120px) + 구역명 + 위험등급 + ← 뒤로가기 버튼
2. 스텝 프로그레스: 원형 번호 4개 (active: 앰버, done: 그린, 대기: 회색) + 연결선
3. 카테고리 헤더: [카테고리 실사 썸네일 48px] + 카테고리명
4. 체크리스트 항목들:
   - 각 항목 탭 → ✅ 적합 (초록 체크)
   - 🚩 버튼 클릭 → 🚩 위반 (빨간 표시)
   - 다시 탭 → 해제
5. 하단: [← 이전] [다음 →] 또는 [🏁 점검 완료] 버튼

각 구역별 4개 카테고리 구성:
- 굴착: 개인보호구(4) → 굴착 장비(4) → 현장 환경(4) → 법규 준수(4)
- 철골: 개인보호구(4) → 철골 장비(4) → 작업 환경(3) → 법규 준수(3)
- 비계: 개인보호구(4) → 비계 구조(4) → 안전 시설(4) → 점검 기준(3)
- 고소: 개인보호구(4) → 추락 방지(4) → 작업 환경(3) → 법규 준수(3)
- 전기: 개인보호구(3) → 전기 안전(4) → 작업 환경(3) → 법규 준수(2)
- 크레인: 개인보호구(3) → 크레인 점검(4) → 작업 반경(4) → 법규 준수(2)

[점검 완료 → 결과 바텀시트]
- 구역 이미지 배너 (120px) + 구역명 + 등급 텍스트
- 원형 점수 링 (SVG, 애니메이션 fill)
  - ≥80: 🟢 안전 (그린), ≥50: 🟡 주의 (앰버), <50: 🔴 위험 (레드)
- 점수 산출 공식: 기본점수(적합/전체×100) - 위반 패널티(위반/전체×40) - 미점검 감점(미점검/전체×10)
- 요약 카드 3개: ✅적합, 🚩위반, ⬜미점검 (각 개수)
- 위반 사항 리스트 (있을 때만 표시)
- 하단: [📋 기록 저장] [다시 점검] 버튼

[📋 기록 탭]
- 히어로 배너 (120px, Unsplash 안전모 사진)
- 통계 3칸: 총 점검 / 안전 통과 / 위험 발견
- 점검 기록 카드 리스트:
  - [구역 썸네일 72px] + 구역명 + 날짜 + 태그(등급, 위반 건수, 적합 수) + 점수 필
- 빈 상태: "🏗️ 점검 기록이 없어요" 안내

[데이터 저장]
- sessionStorage로 점검 기록 저장 (최대 10건)
- 새 점검 완료 시 앞에 추가 (최신순)

[참고]
- 첨부된 safecheck-design.html이 완성된 UI 디자인 레퍼런스
- 이 HTML과 동일한 룩앤필로 React 컴포넌트 구현
- 모든 이미지는 Unsplash URL 그대로 사용 (API 키 불필요)
```
